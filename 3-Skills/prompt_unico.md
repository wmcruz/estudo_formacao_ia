Implemente um painel de posts no frontend e backend existentes:

1. **Backend**: Crie endpoints para:
   - Buscar post por ID: `/posts/{id}` (fonte: `https://jsonplaceholder.typicode.com/posts/{id}`)
   - Listar todos os posts: `/posts` (fonte: `https://jsonplaceholder.typicode.com/posts`)

2. **Frontend**: 
   - Campo de busca por ID do post com exibição do resultado
   - Tabela paginada com todos os posts
   - Tema Material Design com cores sólidas
   - Buscar dados apenas no backend

Use a API gratuita do JsonPlaceholder. Frontend em Angular, backend em Spring Boot.
